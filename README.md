# Spectra 📷 | Analogue Soul Light Meter

Spectra is a premium, pure-web light meter application designed with an "Analogue Soul". It is heavily inspired by classical manual cameras (like the Nikon FM2) and the digital-analogue hybrid philosophy of the Fujifilm X-Series.

Built strictly with HTML, CSS, and Vanilla JavaScript with no external dependencies. 

## ✨ Key Features

- **Viewfinder OSD**: Professional-grade 'A', 'SS', and 'ISO' top info-bar overlaid on a live camera feed.
- **3D Thumb-wheels**: Stacked control room featuring tactile, mechanical 3D scrolling thumb-wheels.
- **Radial Film Selector**: Edge-triggered curved selection dial for classic film simulations (CHROME, VELVIA, ASTIA, ACROS, PROVIA).
- **Auto Exposure Engine**: Experience true photographic mechanics. Set your dial to `AUTO`, and the exposure engine mathematically solves for the perfect exposure (A/S/P/M Modes).
- **Precise Composition**: A Rule of Thirds grid and real-time orientation spirit level.
- **Haptic Feedback**: Mechanical scrolling ticks leveraging the device Vibration API so you *feel* every click of the dial.

## 🚀 Usage

Simply open `index.html` in any modern mobile browser.
No backend or server is required. **Spectra** runs entirely locally in your browser.
